The GGL Registry Editor (a.k.a. the GGL Register or Registry System) is a lightweight and compatible registry editor built in Python. üêç

It features:
1. Ability to write registries in 'archives'
2. Create new archives, export all archive data to TXT/JSON
3. Encryption via AES-256
4. Write a maximum of 25,000 entries in each archive file. An archive can contain unlimited amount of archive files
5. Distinguishable archive file names & extensions: __uncompressed_save{INDEX}.usarch for unencrypted files, and __compressed_save{INDEX}.usarch2 for encrypted files.
6. Searching & filtering entries
7. Archive encryption & password locking
8. Export archives to ZIP and import from ZIP
9. Very customizable, most of the features can be modified in the 'core/app.json' file. (including maximum number of entries, encryption iterations and more.)
